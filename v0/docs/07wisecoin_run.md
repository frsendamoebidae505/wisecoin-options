# 07wisecoin_run.py - 开发文档

## 文件概述

WiseCoin 定时调度执行脚本，在交易日期按配置的时间点自动执行一键分析脚本。

## 作者

playbonze

## 功能描述

1. 判断当前是否为交易日期
2. 在配置的时间点自动触发执行
3. 使用冷却时间防止重复触发
4. 异步调度循环运行

## 核心函数

### `is_trading_day() -> bool`

判断当前是否为交易日期。

**交易日期范围：**
- 周一至周五：全天都是交易日（包括夜盘）
- 周六凌晨 00:00-02:30：算作交易日（周五夜盘延续）
- 周六 02:30 之后至周一：非交易日

```python
def is_trading_day():
    now = datetime.datetime.now()
    weekday = now.weekday()  # 0=周一, 6=周日

    if weekday in [0, 1, 2, 3, 4]:  # 周一至周五
        return True

    if weekday == 5:  # 周六
        if datetime.time(0, 0) <= current_time <= datetime.time(2, 30):
            return True
        return False

    return False  # 周日
```

### `is_scheduled_time(current_time, tolerance_minutes=0) -> tuple`

检查当前时间是否为配置的触发时间点。

**参数：**
- `current_time`: 当前时间
- `tolerance_minutes`: 时间容差

**返回：**
- `(是否为触发时间, 匹配的时间点字符串)`

### `execute_script(script_path) -> bool`

执行目标脚本。

**特性：**
- 根据文件扩展名选择执行方式
- 禁止标准输入防止阻塞
- 实时显示关键输出
- 成功标记检测（避免 `.command` 尾部 read 导致的退出码非0问题）

```python
if ('脚本执行成功' in line) or ('执行成功' in line and '耗时' in line):
    success_marker_seen = True
```

### `scheduler_loop()`

定时调度主循环（异步协程）。

**流程：**
1. 等待检查间隔
2. 判断是否为交易日期
3. 判断是否为触发时间点
4. 检查冷却时间
5. 满足条件时执行脚本

## 异步架构

```python
async def main():
    await scheduler_loop()

if __name__ == '__main__':
    asyncio.run(main())
```
