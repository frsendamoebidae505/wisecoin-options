# 52wisecoin_options_position_client_mock.py - 开发文档

## 文件概述

模拟持仓客户端处理模块，处理模拟持仓的客户端相关操作。

## 作者

playbonze

## 功能描述

1. 读取模拟持仓数据
2. 整合行情信息
3. 计算模拟盈亏
4. 生成持仓报告

## 使用方式

```bash
python 52wisecoin_options_position_client_mock.py
```

## 依赖模块

- `pandas`/`numpy`: 数据处理
- `tqsdk`: 天勤量化SDK

## 注意事项

- 需要先运行模拟持仓生成脚本
- 用于模拟交易分析和报告
