# 00wisecoin_options_backup_clean.py - 开发文档

## 文件概述

期权数据备份目录清理工具，用于清理 `wisecoin_options_backup` 目录中不符合保留条件的备份文件夹。

## 作者

playbonze

## 功能描述

清理备份目录，只保留特定时间点（_0940 和 _2040 结尾）的备份文件夹，其他备份移动到系统回收站。

## 核心函数

### `move_to_trash(target_path)`

将目标目录移动到系统回收站。

**参数：**
- `target_path`: 要删除的目录路径

**返回：**
- 移动后的回收站路径

**处理逻辑：**
1. 获取系统回收站路径 (`~/.Trash`)
2. 如果目标路径已存在同名文件，添加时间戳和计数器后缀

### `clean_backup_directories(backup_root_dir)`

清理备份目录的主函数。

**参数：**
- `backup_root_dir`: 备份根目录路径

**保留规则：**
- 只保留目录名以 `_0940` 或 `_2040` 结尾的备份

## 配置参数

```python
keep_suffixes = ("_0940", "_2040")
```

这两个时间点对应：
- `_0940`: 上午 9:40 的备份（日盘开盘后）
- `_2040`: 晚上 20:40 的备份（夜盘开盘后）

## 使用方式

```bash
python 00wisecoin_options_backup_clean.py
```

## 依赖模块

- `os`: 文件路径操作
- `shutil`: 目录移动
- `datetime`: 时间戳生成

## 注意事项

- 文件被移动到回收站而非永久删除，可恢复
- macOS 系统专用（使用 `~/.Trash`）
- 备份根目录不存在时会给出警告并退出
