# 03wisecoin-options-analyze.py - 开发文档

## 文件概述

期权深度分析与策略筛选模块，执行期权多因子评分、策略筛选和参考数据生成。

## 作者

playbonze

## 功能描述

基于期权行情数据进行深度分析，包括：
- 期权估值分析
- 策略筛选与评分
- 多因子综合评价
- 期权参考数据生成

## 核心功能

**注意：此文件较大（超过 50000 tokens），以下是关键功能概述：**

### 期权分析流程

1. 读取期权行情数据
2. 计算各项分析指标
3. 筛选符合条件的期权合约
4. 生成期权参考数据

### 输出文件

| 文件名 | 说明 |
|--------|------|
| wisecoin-期权参考.xlsx | 期权参考数据（含评分和筛选结果） |

### 运行模式

与其他脚本保持一致的运行模式配置：
- RUN_MODE = 2：TqKq 快期模拟

## 依赖模块

- `tqsdk`: 天勤量化SDK
- `pandas`/`numpy`: 数据处理
- `openpyxl`: Excel操作
- `asyncio`: 异步编程

## 使用方式

```bash
python 03wisecoin-options-analyze.py
```

## 注意事项

- 需要先运行 `01wisecoin-options-ranking.py` 获取行情数据
- 输出完成标志：`期权参考数据生成完成`
